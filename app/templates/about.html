🤖 About ClassForge

ClassForge is a student allocation system built to help educational institutions intelligently group students into classes using AI.

🎯 Our Mission

To simplify and improve the process of class formation by using intelligent algorithms that take both academic performance and student wellbeing into account.

🔬 Technology Behind

We use Graph Neural Networks (GAT), optimization algorithms, and dynamic visualizations powered by Python, Flask, NetworkX, and Matplotlib.

Built with ❤️ by passionate developers and data scientists.
